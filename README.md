Devfiles-Redirect
===================

A small Rack application that redirects `devfil.es` 'mini' urls to 
`devfiles.co`

```
http://devfil.es/4KV8A6cy
       |
 (redirects to)
       |
       -->  http://www.devfiles.co/download/4KV8A6cy/Instagram-51.apk

```

Hosted for free, thanks to [Heroku](https://heroku.com/)
